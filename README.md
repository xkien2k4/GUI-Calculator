#  GUI Calculator (Windows Style)

##  Giới thiệu

**Calculator ** là ứng dụng máy tính để bàn được xây dựng bằng Python, lấy cảm hứng từ giao diện của Windows Calculator.
Ứng dụng cung cấp giao diện hiện đại (dark mode), thao tác mượt mà và hỗ trợ nhiều phép tính cơ bản.

---

##  Mục tiêu dự án

* Xây dựng ứng dụng GUI bằng Python
* Tái hiện giao diện giống Windows Calculator
* Áp dụng thư viện PyQt6 để tạo UI chuyên nghiệp
* Tăng trải nghiệm người dùng với thiết kế hiện đại

---

##  Công nghệ sử dụng

* **Python 3**
* **PyQt6** – xây dựng giao diện người dùng
* **Qt Designer (optional)** – thiết kế UI (nếu có)

---

##  Tính năng chính

*  Các phép toán cơ bản: +, −, ×, ÷
*  Hỗ trợ số thập phân
*  Các phép nâng cao:

  * Bình phương (x²)
  * Căn bậc hai (√x)
  * Nghịch đảo (1/x)
*  Đổi dấu (+/-)
*  Xóa toàn bộ (C), xóa từng ký tự (⌫)
*  Giao diện dark mode hiện đại
*  Nút "=" nổi bật giống Windows

##  Cài đặt & chạy chương trình

### 1. Clone project

```bash
git clone https://github.com/your-username/calculator-pro.git
cd calculator-pro
```

### 2. Cài thư viện

```bash
pip install PyQt6
```

### 3. Chạy chương trình

```bash
python app.py
```
